# 100 Days of Python Backend

Welcome to the 100 Days of Python Backend repository! This repository contains daily projects and exercises to master Python backend development.

## Repository Structure

The projects are organized by day:

- **Day-01-Number-Guessing-Game**: A simple number guessing game.
- **Day-02-Password-Generator**: A script to generate secure passwords.
- **Day-03-Smart-Age-Calculator**: A calculator for age with quizzes and countdowns.
- **Day-04-Even-Odd-Prime**: Logic for checking even, odd, and prime numbers.
- **Day-05-Student-Grade-System**: A Python-based student grading system.
- **Day-06-Quiz-API**: An API project for quizzes.

There is also a separate **Job-Recommendation-Agent** directory containing a specialized backend project.

Happy coding!
