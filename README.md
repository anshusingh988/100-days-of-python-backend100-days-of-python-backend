# Day 2: Password Generator (Python)

This is a beginner-level Python project that generates strong random passwords based on user input.
It is part of my #100DaysOfCode journey to strengthen backend development skills.

## Features
- Generates secure random passwords
- Uses letters, digits, and special characters
- Accepts interactive input for password length

## How to Run
1. Open terminal (PowerShell or CMD)
2. Navigate to folder:
   cd "C:\Users\YourUsername\OneDrive\Desktop\100-days-of-python-backend\Day-02-Password-Generator"
3. Run the Python file:
   python password_generator.py
